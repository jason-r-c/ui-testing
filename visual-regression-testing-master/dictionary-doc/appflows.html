appflows

This documentation describes the Gherkin Steps that can be used when writing Gherkin feature files. Each heading 'Step' is a step phrase that can be reused / copied into a Gherkin feature. A Gherkin Step is bound to its correpsponding Glue Code step.

Each 'Description' heading should inform the reader of what that step does, enabling the user to select the appropriate choice.

Note: Where you see (.*) this represents any predefined value

Step:

I click next

Description:

Use this step for clicking next button/link in an appflow.

Step:

I enter (.*) in the (.*) field

Description:

Use this step for entering values into input fields.

Example usage:
I enter Ferrari in the Surname field

Step:

I select (.*) from the (.*) drop down list

Description:

Use this step for selecting values from drop down lists.

Example usage:
I select Accept Installation Request from the drop down list

Step:

I enter (.*) (.*) (.*) in the (.*) datepicker drop down list

Description:

Use this step for selecting aOS styled date selector elements.

Example usage:
I enter 01 jan 1945 in the Date of Birth datepicker drop down list

Step:

I enter (.*) in the (.*) text area

Description:

Use this step for entering values into text area fields.

Example usage:
I enter additional information in the Installation Day Notes text area

Step:

I check the (.*) checkbox

Description:

Use this step for checking aOS styled checkboxes.

Example usage:
I check the Please tick the box to confirm acceptance checkbox

Step:

I select the (.*) selectable item in the (.*) appflow

Description:

Use this step for selecting aOS styled single item selects.

Example usage:
I select the Customer selectable item in the Select customer appflow

Step:

I confirm (.*) is visible

Description:

Use this step for checking a screen is visible / displayed.

Example usage:
I confirm Finished is visible

Step:

I add (.*) (.*) number in the (.*) telephone field

Description:

Use this step for entering a telephone number (which uses 5 numbers for the first segment and 6 numbers for the second segment).

Example usage:
I add 01234 567890 number in the Mobile Number telephone field

Step:

I select (.*) in the (.*) multi select list

Description:

Use this step for selecting an item in a multi select list.

Example usage:
I select John Ferrari in the Client multi select list

Step:

I select (.*) from the drop down list in the sidebar appflow for All Available (.*)

Description:

Use this step for selecting a drop down item from the All Available tasks / reports in the dashboard sidebar.

Example usage:
I select Manage Hub State from the drop down list in the sidebar appflow for All Available Tasks
